# Roadmap

## Release 0.1

- Single-owner fork and access gate
- OAuth, encrypted token storage and deletion controls
- Persistent drafts, threads, schedules and analytics
- Feed-derived ideas and reply opportunities
- Human-reviewed publishing and policy-gated AI
- Import/export, REST API and MCP

## Next

- Media, polls and X Articles
- Better topic clustering and multilingual voice analysis
- Queue drag-and-drop and timezone-aware recommendations
- Backup adapters and encrypted off-site backups
- Optional local-model providers

## Explicitly out of scope

- Unsolicited DMs
- Autonomous replies or mentions
- Scraping or browser automation against X
- Engagement manipulation
- Shared multi-tenant credential custody
